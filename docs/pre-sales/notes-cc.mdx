title: Notes for clients paying with debit/creditcards
sidebar_label: Notes for CC/DB customers
---

We support a large list of credit/debitcards but please note that we only accept cards that support 3D Secure, you have to pass the 3D Secure check to pay using a card on our system.

Any payments processed via a VPN will also be blocked automatically.

**Also note that it is not possible to pay with any kind of proxy/vpn service!**
